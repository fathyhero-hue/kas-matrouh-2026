المهم:
الملفات لا تعمل بمجرد تحميلها. لازم تستبدل ملفات page.tsx الفعلية داخل المشروع.

الطريقة الأسهل:
1) فك هذا الملف المضغوط داخل فولدر المشروع:
   C:\Users\HERO\tournament-app2

2) شغل:
   install-knockout-update.bat

3) اقفل السيرفر القديم:
   Ctrl + C

4) شغل المشروع:
   npm run dev

المسارات التي سيتم استبدالها تلقائيًا:
- app/page.tsx أو src/app/page.tsx
- app/admin/page.tsx أو src/app/admin/page.tsx

النسخ الأصلية ستتخزن في:
backup-before-knockout

بعد التشغيل، افتح:
- الصفحة الرئيسية: http://localhost:3000
- صفحة الإدارة: http://localhost:3000/admin

علامات نجاح التحديث:
- في الصفحة الرئيسية/بطولة المثاني يظهر تبويب: الأدوار الإقصائية
- في صفحة الإدارة/المباريات، عند اختيار دور الـ 16 تظهر إعدادات الأدوار الإقصائية:
  كود المباراة الحالية
  المباراة القادمة للفائز
  مكان الفائز
  الفريق الصاعد عند التعادل/الترجيح
